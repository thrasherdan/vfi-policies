# Privacy Policy — VFI Reels Recycler

_Last updated: July 2, 2026_

VFI Reels Recycler is an internal content-publishing tool operated by Virtually Funny Inc. ("VFI") to manage video publishing and analytics for VFI's own Facebook Page (facebook.com/danielthrasherofficial) using Meta's official APIs.

## What this app does

- Publishes videos (reels and stories) to VFI's own Facebook Page
- - Reads engagement insights for content on that Page
 
  - ## Data we collect
 
  - This app accesses only data belonging to VFI's own Facebook Page: post metadata and aggregate performance insights provided by Meta's Graph API. It does not collect, store, or process personal data from other Facebook users. No data is sold, shared, or transferred to third parties.
 
  - ## Data storage and deletion
 
  - Page content metadata and aggregate insights are stored on VFI's private infrastructure. To request deletion of any data, contact us and we will delete it within 30 days.
